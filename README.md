# Python Escape Room Game

A console-based Escape Room game built with Python.

## Features

* Math Puzzle Door
* Secret Word Guessing Door
* Lucky Number Door
* Lives System
* Badge Rewards
* Multiple Levels

## Technologies Used

* Python
* Random Module
* Functions
* Loops
* Conditional Statements
* Sets

## How to Run

```bash
python escape_room.py
```
